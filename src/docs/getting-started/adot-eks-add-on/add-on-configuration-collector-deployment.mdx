As discussed in the section [Add-on Advanced Configuration](/docs/getting-started/adot-eks-add-on/add-on-configuration), 
EKS add-ons provides the ability to configure ADOT Collector(s) to be deployed during add-on installation.

Using the EKS add-on advanced configuration, you can deploy ADOT Collector(s) as preconfigured custom resources,
each with its own set of configurable values. It is possible to deploy multiple preconfigured 
custom resources together in one advanced configuration using the EKS add-on.

The supported preconfigured custom resources are listed below:
## [Prometheus Metrics](/docs/getting-started/adot-eks-add-on/config-prometheus-metrics)
## [OTLP Ingest](/docs/getting-started/adot-eks-add-on/config-otlp-ingest)
## [Container Logs](/docs/getting-started/adot-eks-add-on/config-container-logs)

Use the links above to learn about the available configuration options.
